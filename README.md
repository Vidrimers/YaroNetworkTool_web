# Yaro Network Tool Portal

Веб-портал для управления VPN-подписками. Express SPA, проксирует запросы к VPS Management API.

## Архитектура

```
Браузер → Portal (Express :3000) → VPS API (:333) → SQLite DB
                ↕
          Telegram OAuth / JWT
```

Portal **не хранит данные** — только фронтенд + прокси.

## Функционал

### Пользователь
- Дашборд: статус, трафик (день/неделя/месяц), подписка, устройства
- Подключение: ссылка + QR-код для VPN-клиентов
- Продление: запрос на продление подписки
- История запросов, загрузка клиентов, справка

### Админ (telegram_id=137981675)
- Управление Xray (старт/стоп/рестарт/логи)
- Клиенты: CRUD, модалка с полной информацией
- Мониторинг: баны, трафик, устройства, торренты
- Запросы на продление: одобрение/отклонение
- WARP-управление

## Установка

```bash
npm install
cp .env.example .env
# заполнить .env реальными ключами
npm start
```

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `PORT` | Порт сервера (по умолчанию 3000) |
| `VPS_API_URL` | URL Management API на VPS |
| `VPS_API_KEY` | API-ключ для VPS |
| `TELEGRAM_BOT_TOKEN` | Токен Telegram-бота |
| `TELEGRAM_ADMIN_ID` | Telegram ID администратора |
| `JWT_SECRET` | Секрет для JWT-токенов |

## Деплой

Шаблон Nginx в `nginx.conf` — заменить `YOUR_DOMAIN` на реальный домен и настроить SSL (Let's Encrypt).

## Стек

- Node.js 18+
- Express
- ES modules
